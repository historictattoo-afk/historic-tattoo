# Historic Tattoo — Portland, OR

New website for Historic Tattoo built with Next.js 14, TypeScript, and Tailwind CSS.

## Pages

- `/` — Home (hero, services preview, artist grid, visit CTA)
- `/about` — Shop history and info
- `/artists` — Full artist directory with Instagram links
- `/get-what-you-get` — GWYG / Plinko service pages with FAQ

## Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Oswald (display), Inter (body), Playfair Display (serif accents) via next/font/google
- **Hosting**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

This project is configured for one-click deployment on Vercel. Connect your GitHub repo at
[vercel.com](https://vercel.com) and it deploys automatically on every push to `main`.
